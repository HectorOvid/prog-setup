# Lazyvim setup overview

## Font
-> Font needs to be installed, for proper small icons in nvim

```
mdkir ~/.fonts
cd ~/.fonts
wget -c https://github.com/ryanoasis/nerd-fonts/releases/download/v3.1.1/RobotoMono.zip
unzip RobotMono.zip
rm RobotoMono.zip
```
